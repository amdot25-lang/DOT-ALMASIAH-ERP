# v26 — غرفة القيادة

- تحويل الصفحة الرئيسية للمالك إلى غرفة قيادة تنفيذية.
- أربع مناطق رئيسية: الحالة المالية، المبيعات والهدف، المخزون، وما يحتاج الانتباه.
- إضافة تقييم عام لأداء الوكالة من 100.
- إضافة ملخص مدير تفسيري.
- إضافة توقع نهاية الشهر والمطلوب يوميًا.
- إضافة تقدير مدة كفاية المخزون.
- إضافة رسم مبسط لحركة الألماس خلال 7 أيام.
- إبقاء التفاصيل التشغيلية السابقة داخل زر مخصص.
- عدم حذف أو تغيير أي بيانات أو قسم قائم.
