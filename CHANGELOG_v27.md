# v27 — الإدارة الذكية

- إضافة مركز تنبيهات ذكي داخل مركز القيادة ولوحة الأداء.
- تطوير سجل النشاط الموجود والمحافظة عليه.
- إضافة بحث شامل ثابت في العمليات التاريخية والسحابية.
- إضافة لوحة أداء: أعلى عملية، أفضل يوم، المتوسط، أكثر وسيلة دفع، وإجماليات التشغيل.
- إضافة نسخة احتياطية ذكية تجمع بيانات المتصفح مع لقطة قراءة من البيانات السحابية.
- استعادة النسخة الذكية آمنة وتعيد البيانات المحلية فقط دون حذف بيانات السحابة.
- تغيير الاسم إلى «مركز القيادة».
- عدم إضافة فواتير أو عملاء أو مدفوعات؛ سلة تبقى النظام الأساسي لهذه الوظائف.
