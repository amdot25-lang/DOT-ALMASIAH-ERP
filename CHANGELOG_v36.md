# v36 — الواجهة الاحترافية المستقرة

- نقل بطاقة اسم المستخدم والحساب إلى أعلى اليسار بصورة مستقلة وأنيقة.
- تثبيت زر القائمة في أعلى اليمين ومنع التداخل على الجوال.
- حذف شريط: تصدير التقرير الإداري، تقرير المدير، تقرير واتساب، وإجراءات إضافية.
- عدم المساس بفلترة لوحة التحكم أو دائرة هدف الألماس أو بطاقات الهدف أو الحسابات.
- عدم إضافة مركز القيادة التنفيذي أو التقارير إلى قائمة المالك.
- لم تُعدّل ملفات Firebase أو Firestore أو البيانات التاريخية أو النسخة الاحتياطية.
