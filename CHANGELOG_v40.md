# v40 — لوحة المشرف للجوال

- تصميم لوحة مشرف خفيفة ومخصصة للجوال.
- إضافة عدد عمليات البيع ضمن مؤشرات الفترة.
- إضافة متوسط قيمة العملية بالماس.
- عرض ألماس الفترة وقيمة المبيعات والعمليات المعلقة ونسبة الهدف.
- زر كبير وواضح لتسجيل بيع جديد.
- بطاقة مخزون مع تقدير تقريبي لعدد الأيام المتبقية.
- عرض آخر خمس مبيعات دون أسماء المشترين أو أي بيانات شخصية.
- الإبقاء على فلترة يوم/شهر/سنة.
- عدم تغيير صلاحيات المشرف أو بيانات Firebase وFirestore.
